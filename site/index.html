rdpweb

Browser-based RDP client. One binary, self-contained frontend.

Quick install (Linux & macOS)

curl -fsSL https://andrewtheguy.github.io/rdpweb/install.sh | bash

Downloads the release tarball for your platform, verifies its SHA-256 against the GitHub-published digest, and installs to /usr/local/opt/rdpweb with a rdpweb launcher on your PATH. You may be prompted for sudo to write under /usr/local.

Then

# edit your RDP target + credentials
$EDITOR /usr/local/opt/rdpweb/current/etc/rdpweb.env

# start the server
rdpweb serve

Options

  • Install a specific release: … | bash -s -- v0.1.0
  • Custom location: PREFIX=$HOME/.local/opt/rdpweb BINDIR=$HOME/.local/bin … | bash

Each install keeps the previous version under versions/ so you can roll back by repointing the current symlink.

Source & releases on GitHub. Review the install script before piping to a shell.
